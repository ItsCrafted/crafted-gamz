How To Make Your Own Deploy Of Crafted Gamz:

Step 1: Requirements

You must have:

  • A Github account
  • A Netlify account
  • A custom domain (paid) (recommended from Namecheap) ONLY IF YOUR SCHOOL BLOCKS "example.netlify.app" DOMAINS (ex. https://crafted-gamz.netlify.app/) In the sense that you DO need a custom domain, you'll need a Namecheap account

Step 2: Repo Cloning

Make sure you are signed into Github

"repo" = Repository

Go to https://github.com/ItsCrafted/crafted-gamz

I recommend starring the repo to pin it to your account

Click "fork" (top right of repo section, next to "Unwatch" and "Star"

Make the repo name "crafted-gamz-fork"

Select "Copy the main branch only"

Click "Create fork"

Wait for the fork operation to complete, so your new repo looks something like https://github.com/ItsCrafted/crafted-gamz

Step 3: Deploying

Go to your Netlify "Projects" Page

Click "Add new project" then "Import an existing project"

Where it says "Let’s deploy your project with", Select Github

Verify Netlify access to Github

Select "crafted-gamz-fork" from the list

Set the following fields to their respective values:

  • Team is not needed
  • Project name should be set to whatever you wish (this will determine the value before ".netlify.app")
  • Branch to deploy should be set to "main"
  • Base directory should be blank
  • Build command should be blank
  • Publish directory should be set to "." (as in a period)
  • Functions Directory should be set to "netlify/functions"
  • Add NO Environment variables

Click "Deploy "____"

Wait for it to go live (takes around 15 minutes)

Please Email "crafted@craftedgamz.com" the url to your new site

Step 4: Adding A Custom Domain From Namecheap (If Needed)

Go to your Namecheap Dashboard

Click on "Manage", on the domain you want to add

Go to "Advanced DNS"

Delete All existing Host Records

Go back to the "Domain" tab

Scroll down to "Nameservers"

Click Custom DNS

Open Netlify in another tab, this time in "Domains" (sidebar)

Click "Add or register domain", then "Add a domain you already own"

Enter your domain from Namecheap into "What domain would you like to use?", then "Verify"

It should say "_____ is already registered. If you registered this domain name through a different registrar, select Add domain to begin delegating it to Netlify"

Click "Add domain"

Add no DNS records, just hit "Continue"

Go back to your Namecheap tab

Add 4 Nameservers as the following, making sure that they are the only 4 Nameservers

  • dns1.p03.nsone.net
  • dns2.p03.nsone.net
  • dns3.p03.nsone.net
  • dns4.p03.nsone.net

Click the check mark

Go back to Netlify

Click "Done"

I recommend clicking "Enable IPv6"

Go back to your "Projects" tab

Click "Domain management" (other tab bar)

Click "Add domain alias"

Enter your custom domain (no https or www), ex. "craftedgamz.com"

Wait until it says "Netlify DNS"

Your domain now links to Crafted Gamz!

Wait 24 hours for Netlify to add https
